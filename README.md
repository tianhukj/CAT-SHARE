 # 项目简介
 CAT-SHARE 工具是一款基于 Chrom 浏览器开发的插件，它可以在网站中选中文字后生成漂亮的分享卡片，并保存到本地。
 # 快速开始
 > [!IMPORTANT]
 > 此步骤非常重要
 - **下载ZIP包**
 - **打开`Crome`浏览器**
 - **打开开发者模式**
 - **将插件ZIP包拖到`插件管理`中**
 - **在网站中选择文字后即可使用**
